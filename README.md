# nasa
Repository dedicata agli esercizi del corso Media&amp;Interaction SP 2026

### Idea di base 
L'idea di partenza è quello di sviluppare un visualizzatore d'immagini del suolo lunare, 
esteticamente presentato come il suolo lunare, riprendendo le forme del cratere tramite Heightmapping.

L'idea può espandere e diventare un visualizzatore piu grandi dei suoli di vari pianeti sempre presentandoli mediante Heightmapping qualora possibile.
